# Learning Tracker

At first, I built this project to track my own learning progress while studying programming. Later, I turned it into a portfolio project and expanded it by implementing everything I learned.

The app allows users to add topics, track confidence levels, log study hours, and view their overall progress.

This is my main portfolio project while learning full-stack development.

## Why I built this

I wanted a way to actively track what I’m learning and measure progress over time, instead of just passively studying.

## Features

- User authentication (register & login with JWT)
- Add, update, and delete learning topics
- Track confidence levels (from 0–5)
- Log study hours and last practiced date
- Filter topics by name, confidence, and mastery
- View overall statistics and progress

## What I learned

- Building a full-stack app using React and Node.js
- Implementing authentication with JWT
- Handling real-world issues like CORS and deployment
- Structuring a backend with Express and MongoDB
